# git 필기
## 커밋/ 푸시 규칙
* 꼭! 항상! 절대! `add`, `commit`, `push`는 최상단 폴더에서 작업하기
## 처음으로 깃을 시작할 때 한 번만 하면 되는 작업

### `git init`

### `git remote`
* `git remote add origin [나의 깃헙의 원격저장소(github) 레포 주소]`

* `git remote -v`: 내가 등록한 원격저장소 레포 주소 확인

* `git remote remove origin` : 등록한 원격저장소 삭제
