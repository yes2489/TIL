# TCP/IP 4계층
| 가장 널리 사용되는 인터넷 통신 프로토콜

## 네트워크 연결 계층 (Network Access)
 - OSI 7계층: 물리 계층, 데이터 링크 계층
 - 프레임 단위 전송 담당
 - 같은 네트워크(LAN) 내에서만 유효
 - MAC 주소 활용
 - 대표 기술: Ethernet, ARP

## 인터넷 계층 (Internet)
 - OSI 7계층: 네트워크 계층
 - 다른 네트워크 간의 데이터 전송 담당
 - IP 패킷을 전송하고 라우팅 함

## 전송 계층 (Transport)
 - OSI 7계층: 전송 계층
 - 포트 번호를 통해 프로세스 식별
 - 흐름 제어, 혼잡 제어(TCP)
 - 신뢰성 여부에 따라 TCP/UDP로 구분

## 응용 계층 (Application)
 - OSI 7계층: 응용 계층, 표현 계층, 세션 계층
 - 사용자 관점에서 실제 서비스 제공
 - 데이터 형식, 세션 관리 포함
